MyFilter

Description

Filter a list using fold. In the Prelude, filter is defined as a recursive function:

  filter :: (a -> Bool) -> [a] -> [a]
  filter _pred []    = []
  filter pred (x:xs)
    | pred x         = x : filter pred xs
    | otherwise      = filter pred xs
